# dial-gauge (v.2)

원 또는 다각형 바깥을 접하여 이동하는 원(probe)에 대해서 probe의 중심이 이동하는 궤적

- dial-circle : 내부가 원
- dial-rectangle : 내부가 사각형
  + fix : circle에 대한 코드를 rectangle로 변환
  + moving : probe가 움직이는 애니메이션 구현 
  + single ball : 원점에서 probe까지의 거리를 그래프로 표시
  + dual ball : probe가 두 개인 경우

상기 작업의 역순으로 probe 궤적에 대한 정보가 있을 때 내접하는 도형을 구하는 코드

- dial-polygon : 내부가 다각형 또는 비정형
